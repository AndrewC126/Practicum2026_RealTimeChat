4. User Presence

US-401 — See who is online

As a user, I want to see which users are currently online so that I know who I can chat with.

Acceptance Criteria:

Each user in the room list or member list has a visible online/offline status indicator (e.g. green dot)
The indicator updates in real time when a user connects or disconnects
Users who log out are marked offline immediately
Users who close the browser tab are marked offline within 10 seconds


US-402 — See room members

As a user in a chat room, I want to see who else is in the room so that I know my audience.

Acceptance Criteria:

A member list panel shows all users currently in the active room
Online members are listed before offline members
The list updates in real time as users join or leave
Clicking a username shows basic profile info (username, join date)
